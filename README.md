# Topic
코스피 200 구성 기업들이 '배당액 공시 이후에 배당기준일을 지정하는 제도를 도입했는지' 여부를 확인하는 코드 구현.

이 저장소에는 KRX에서 코스피 200 구성 종목을 받아온 뒤, DART 공시를 검색해 배당 기준일 지정 제도 도입 여부를 추정하는 스크립트가 포함돼 있습니다.

## 준비물
- Python 3.11 이상
- DART Open API 인증키: `DART_API_KEY` 환경 변수로 제공

## 사용 방법
1. 필요 시 가상환경을 생성하고 활성화합니다.
2. DART API 키를 환경 변수(`DART_API_KEY`)로 설정하거나 실행 시 `--dart-api-key` 인자를 넘깁니다.
3. `python main.py --start-date 20210101 --end-date 20241231 --output result.json`처럼 실행합니다.
   - 시작/종료일을 지정하지 않으면 최근 3년치 공시를 검색합니다.
   - `--output`을 생략하면 결과 JSON을 표준출력으로 보여줍니다.

스크립트는 다음 단계를 수행합니다.
1. KRX `MDCSTAT00601` 엔드포인트에서 코스피 200 구성 종목을 조회
2. DART 법인코드 파일을 내려받아 KRX 종목코드와 매칭
3. 정관변경·배당 관련 공시(B001, I001)를 키워드(`배당기준일`, `정관변경` 등)로 필터링해 제도 도입 여부 추정
